# Tacoma Dome Link

!!! warning "Stub article"
    This is a placeholder for a planned line. Details and dates will be filled in as Sound Transit's plans firm up.

A planned extension of the [1 Line](line-1.md) south from **Federal Way Downtown** — the current southern end — through Fife to the **Tacoma Dome**, where it would meet Sounder commuter rail and the T Line streetcar.

**Status:** in planning, projected for the early-to-mid 2030s.

**Why it matters:** it would link the region's two largest cities, Seattle and Tacoma, by fast rail, and connect a string of South Sound communities to the network for the first time.

*See the [future lines overview](future.md) for the rest of the planned network.*
